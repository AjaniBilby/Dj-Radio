Info

This internet radio was created for a school assignment.

It has a special auto-play function so if no one queues the next song the server will pick a random song base on listener statistics. It compiles likes, dislikes, and listens for each hour and then spreads each weighting on the hours around it so that if a song is popular at 6pm as time gets closer and closer to 6pm the weighting of that song will increase.

  • Object-manipulation
  • MusicMetaData
  • Mass-Random
  • Throttle
  • Passer
  • Lame
